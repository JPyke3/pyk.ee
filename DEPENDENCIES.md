# Dependencies

This document lists all external libraries used on pyk.ee, per Article XII of the project constitution.

## JavaScript Libraries

### marked.js

- **Version**: 4.3.0
- **Purpose**: Client-side Markdown parsing for blog posts
- **License**: MIT
- **Source**: https://github.com/markedjs/marked
- **File**: `js/vendor/marked.min.js`

### highlight.js

- **Version**: 11.9.0
- **Purpose**: Syntax highlighting for code blocks in blog posts
- **License**: BSD-3-Clause
- **Source**: https://github.com/highlightjs/highlight.js
- **File**: `js/vendor/highlight.min.js`

## CSS

### hljs-terminal.css

- **Purpose**: Custom highlight.js theme matching the amber-on-black terminal aesthetic
- **File**: `css/vendor/hljs-terminal.css`
- **Note**: Custom theme created for this project, based on highlight.js theming guidelines

## Fonts

### JetBrains Mono

- **Version**: 2.304
- **Purpose**: Primary monospace font for terminal aesthetic
- **License**: SIL Open Font License 1.1
- **Source**: https://www.jetbrains.com/lp/mono/
- **Files**: `fonts/JetBrainsMono/*.woff2`
